# Web Public Assets

This directory contains static assets for the web application.

## Structure
- `logo.svg` - Embr logo
- `icons/` - Icon files for favicon, etc.
- `fonts/` - Custom font files (if needed)
- `images/` - Static images

## Adding Files

Place your static files here and reference them in your Next.js components:
```tsx
<Image src="/logo.svg" alt="Embr Logo" />
```
