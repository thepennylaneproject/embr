# Mobile Assets

This directory contains assets for the Expo mobile application.

## Required Files
- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen (1284x2778 recommended)
- `adaptive-icon.png` - Android adaptive icon (1080x1080)
- `favicon.png` - Web favicon (192x192)

## Adding Icons

Use a tool like:
- [Figma](https://figma.com) - Design tool
- [Icon.kitchen](https://icon.kitchen) - Quick icon generator
- [App Logo Creator](https://www.applogomaker.com) - Logo generator

Make sure your icons meet Expo's requirements for proper rendering on iOS and Android.
